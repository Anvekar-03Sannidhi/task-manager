# Smart Task Management System 

A Full-stack web-application built using Django that allows users to manage their daily tasks efficiently.

## FEATURES
- User Authentication (Signup, Login, Logout)
- Create, Update, Delete Tasks (CRUD)
- User-specific task management
- Task filtering (status & priority)
- Search functionality
- Dashboard with task analytics

## TECH STACK
- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite

## Security
- Login required for task access
- User-specific data isolation

## Dashboard
- Total Tasks
- Completed Tasks
- Pending Tasks

## Future Improvements
- UI enhancement using React
- Notifications
- Cloud deployment
